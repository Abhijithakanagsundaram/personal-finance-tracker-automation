Personal Finance Tracker (Low-Code Automation Project)

 Overview
This project is a fully automated Personal Finance Tracker built using low-code tools.

It tracks daily expenses, monitors monthly budgets, and sends automated alerts when spending exceeds defined limits.



Tools Used
- Jotform (Expense Submission)
- Airtable (Database & Dashboard)
- Make (Budget Alert Automation)
- Gmail (Email Notifications)
- n8n (Real-time Trend Updates)
- Lumen5 (Project Demo Video)
